# Movie Ticket Booking System

A simple console-based movie ticket booking system built with Java using TDD.

## Features
- Book movie tickets
- View bookings by customer
- Cancel bookings
- Console-based UI

## Technologies
- Java
- JUnit 5 (Testing framework)
- Mockito (Mocking framework)
- JaCoCo  (Code coverage tool)
- Gradle   
- Docker

## Key Involvements
1. 4+ SUT classes with dependencies  
2. Dependency injection  
3. Mocking with Mockito  
4. Dynamic object creation in 2+ places  
5. Console UI  
6. TDD with RED/GREEN/REFACTOR commits  
7. Code coverage with JaCoCo  

## How to Run
1. Clone the repository
2. Build: `./gradlew build`
3. Run tests: `./gradlew test`
4. Run application: Run App.java
